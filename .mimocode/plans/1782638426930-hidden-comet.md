# Portfolio Redesign Plan — Match artabidkhori.com Style

## Overview
Redesign Mahan Asgari's portfolio from a traditional single-page layout into a modern, minimal dark-themed design matching artabidkhori.com's aesthetic while preserving all existing content.

---

## Files to Modify
- `index.html` — Complete section restructuring
- `css/style.css` — New design tokens, component styles, animations
- `js/main.js` — Nav indicator, FAQ accordion, word reveal, scroll reveals

## 5. Responsive Design

### Breakpoints
- **Desktop (>1024px)**: Floating pill nav, 2×2 service grid, full project rows
- **Tablet (≤1024px)**: Single-column services, smaller project images
- **Mobile (≤768px)**: Hide floating nav (use hamburger), stack everything, hide watermarks
- **Extra Small (≤480px)**: Smaller hero title, stacked stats
